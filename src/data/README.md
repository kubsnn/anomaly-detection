# Training Strategy for Video Anomaly Detection

## Data Distribution

Total dataset: 80 hours
- Normal behavior: ~60 hours (75%)
- Anomalous behavior: ~20 hours (25%)

## Data Split

1. Training Set (70%):
   - Total: ~56 hours
   - Normal: ~42 hours
   - Anomalous: ~14 hours

2. Validation Set (20%):
   - Total: ~16 hours
   - Normal: ~12 hours
   - Anomalous: ~4 hours

3. Test Set (10%):
   - Total: ~8 hours
   - Normal: ~6 hours
   - Anomalous: ~2 hours
