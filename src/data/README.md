# Training Strategy for Video Anomaly Detection

## Data Distribution

Total dataset: 80 hours
- Normal behavior: ~60 hours (75%)
- Anomalous behavior: ~20 hours (25%)

### Data Split for autoencoder training for reproducibility 

1. Training Set (70%):
   - Normal: ~42 hours

2. Validation Set (20%):
   - Normal: ~12 hours

3. Test Set (10%):
   - Normal: ~6 hours
